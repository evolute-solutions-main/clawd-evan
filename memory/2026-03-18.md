## GHL Appointments Discovery

Found how to pull appointments from GoHighLevel:
- Calendar events API returns empty (not used for appointment tracking)
- **Appointments are tracked via tags**: `confirmed_appointment`, `unconfirmed_appointment`
- **Correct endpoint**: `GET /contacts/{contactId}/appointments`
- Returns: appointmentStatus, calendarId, startTime, endTime, assignedUserId, address (meeting link)
- Flow: Query contacts by tag → For each contact, call `/contacts/{id}/appointments` → Filter by date
